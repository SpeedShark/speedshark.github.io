02.esxcli常用命令

发表于 2019-12-02 | 分类于 VMware ， vSphere6.5 ， esxcli | 次阅读

简介

  • 命令行管理vSphere

查看发行版本

  • esxcli system version get

查看时间

  • esxcli system time get

修改时间

  • esxcli system time set -y=2016 -M=9 -d=13 -H=10 -m=9

开关机重启vsphere 6.5

注意该命令需要vsphere 6.5 进入维护模式

  • esxcli system shutdown reboot/poweroff

开启关闭维护模式

  • esxcli system maintenanceMode set –enable true/false

网络接口查询

  • esxcli network ip interface ipv4 get

    查看路由表

  • esxcli network ip route ipv4 list

查看物理网卡列表

  • esxcli network nic list

    启用禁用网卡接口

  • esxcli network nic down/up -n=vmnic2
